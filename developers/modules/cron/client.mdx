This document details the client interfaces for interacting with the Cron module, including CLI commands and gRPC-gateway REST endpoints.

## CLI Commands

The `neutrond` CLI provides commands to interact with the Cron module under the `cron` namespace.

### Query Commands

#### Get all schedules

```bash
neutrond query cron schedules
```

#### Get a specific schedule

```bash
neutrond query cron schedule [schedule-name]
```

#### Get module parameters

```bash
neutrond query cron params
```

### Transaction Commands

#### Create a schedule

```bash
neutrond tx cron create-schedule [name] [period] [messages]
```

#### Update a schedule

```bash
neutrond tx cron update-schedule [name] [period] [messages]
```

#### Remove a schedule

```bash
neutrond tx cron remove-schedule [name]
```

## REST Endpoints

- `GET /neutron/cron/schedules` - Get all schedules
- `GET /neutron/cron/schedule/{name}` - Get specific schedule
- `GET /neutron/cron/params` - Get module parameters
